
## 依赖相关
首先要安装最新版的Node.js，安装成功后输入`node -v`与`npm -v`检查安装是否成功，成功后在本项目目录下执行`npm install`即可，至于组件的依赖项等后续确定再进行补充

node --version 16.17.0

npm --version 8.15.0
